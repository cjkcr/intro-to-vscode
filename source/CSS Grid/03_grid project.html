HTML（Hypertext Markup Language）
用于描述页面的结构
它使用标签的形式来标识网页中的不同级成部分
  • html 根标签（元素），网页中所有的内容都是写在根元素里边.
  • head 网页的头部，其中的内容不会在网页中直接出现，主要是用来帮助浏览器或搜索引擎来解析网页.
  • meta 标签用来充置网页的元数据,这里用来设置网页的字符集等.
  • charset 指定网页的字符集.
  • name 指定的数据的名称.
  • keywords 表示网站的关键字,可以同时指定多个关键字,隔开用逗号.内容可被摸索.
  • description 用于指定网站的描述,显示在搜索引擎的结果中.
  • title 标签的内容会作为摸索结果的超链接上的文字显示.
  • http-equiv='refresh' 用于重定向.
  • content 指定的数据的内容
  • title 显示在浏览器的标题栏,搜索引擎会主要根据title中的内容来判断网页中的主要内容.
属性
  • 可以为HTML标签设置属性
  • 通过属性为HTML元素提供附加信息
  • 属性需要设置在开始标签或自结束标签中
  • 属性总是以 名称/值对 的形式出现
  • 有此属性可以是任意值，有些则必须是指定值
注释
  • HTML注释中的内容不会在网页中显示
  • 合理的使用注释可以帮助开发人员理解网页的代码
  • 注释不能嵌套
字符集
  • 字符集规定了如何将文本转换为二进制编码
  • 常见的字符集：ASKII、ISO8859-1、GBK、GB2313、UTF-8.
乱码
  • 如果我们在保存文件和在浏览器中读取页面时用的不是一个字符集就会有可能出现乱码
  • 为了让页面有良好的使用性，一般使用UTF-8编码。
  • 在HTML5中只需要使用meta标签就可以完成，meta charset='utf-8'
HTML
  • 用于告诉浏览器这个文档中包含的信息是用HTML编写的
  • 其中有二个子标签head和body标签
  • head标签用来表示网页的元数据包含了浏览器和搜索引擎使用的其他不可见信息
  • body标签用来设置网页的主体，所有在页面中能看到的内容都应该编写到body中。
实体
  • 在网页中编写的多个空格会自动解析为一个空格, 我们不能直接书写一些特殊符号,则需要使用实体
  • &实体的名字;
  • & nbsp; 空格
  • & gt; 大于号
  • & lt; 小于号
  • & copy; 版权符号
  • & quot; 引号
  • & reg; 注册商标
  • & trade; 商标
  • & amp; 和符号&
body标签
  • 在网页中HTML专门用业负责网页的结构
  • 所以在使用HTML标签时,应该关注的是标签的语义,而不是它的样式
  • 标题标签:
  • h1-h6 共有六级
  • 重要性递减
  • h1重要性仅次于title标签,一般情况下页面只有一个h1
  • 在页面中独占一行的称为块元素(block element)
  • hgroup标签用来为标题分组,可以将一组相关的标题同时放入到hgroup
  • p标签 表示页面中表示一个段落,也是独占一行的块元素
  • blockquote标签 表示一个长引用,前面会退格,也是块元素
  • br标签 表示换行
    • 在页面中不会独占一行的称为行内元素(inline element)
  • 1.em标签 用来表示语音语调的一个加重
  • 2.strong标签 用来表示强调,重要内容,这也是行内元素
  • 3.q标签 表示一个短引用,加引号的
    块元素(block element)
  • 在网页中一般过过块元素对页面进行布局一般情况下会在块元素中放行内元素,但不会在行内元素放块元素
  • 块元素中基本上什么都能放
  • p元素中不能放任何的块元素
    • 行内元素(inling element)
  • 行内元素主要用来包裹文字
  • 浏览器在解析网页时,会自动对网页中不符合规范的内容进行修正
  • 比如:标签写在了根元素外部,p元素中有块元素
    • 布局标签(结构化语义标签)
  • div 没有语义,就用来表示一个区块,目前来说还是我们主要的布局元素.
  • span 行内元素,没有任何语义,一般用于在网页中选中文字.
    • 以下在网页中用的不多
  • header 表示网页的头部
  • main 表示网页的主体部分(一个页面中只会有一个),它不应包含在文档中重复出现的内容，比如侧栏、导航栏、版权信息、站点标志或搜索表单。
  • nav 表示网页中的导航
  • aside 和主体相关的其它内容(侧边栏),aside 的内容应该与附近的内容相关。
  • article 表示一个独立的文章
  • section 表示一个独立的区块,上边的标签都不能表示时使用
列表()
html列表一共有三种.可以互相嵌套.
    有序列表
  • ol标签
    • 无序列表 这个用的最多
  • ul标签
    • 定义列表
  • dl标签
  • dt 表示定义的内容

超链接
a标签 定义超链是一个行内元素,但是可以嵌套除了它自已之外任何元素
属性:
    href 指定跳转的目标地址
  • 值可以是一个外部网站
  • 也可以是一个内部页面地址
  • 内部地址使用相对路径
    • ./ 表示当前文件所在目录,可以省略不写
    ../ 表示上一级目录
    #id 表示到当前页面标签id处
    # 到当前页面底部,或占位符
    javascript:; 占位符
    target 用来指定超链接的拉开的位置
  • -self 默认值.在当前页面中打开
  • -blank 在一个新的页面中打开
    id 每一个标签都可以添加一个id属性,是元素的唯一标识
img标签用于向当前页面引入一个外部图片:
这是一个自结束标签,属于替换元素(基于块和行内元素之间,具有两种元素的特点)
    属性:
  • src 属性指定外部图片路径
  • alt 图片的描述,无法加载时显示,搜索时识别用
  • wight 图片宽度(单位像素)
  • height 图片高度.只修改一个,则另一个会等比缩放
    • 注意:
  • pc端不建议修改图片大小,需要多大就栽多大
  • 移动端,经常需要对图片上缩放(大图缩小)
    • 图片格式:
  • jpg 颜色丰富,不支持透明效果,不支持动图,一般用来显示照片
  • gif 颜色较少,支持简单透明,支持动图
  • png 颜色丰富,支持复杂透明,不支持动图(专为网页而生)
  • webp goole推出网页专用格式,有所有优点,还特别小,但兼容性不好
  • base64 将图片使用base64编码,将图片转为字符,通过字符形式导入图片 这样可以使图片和网页一起加载,src='base64'
iframe标签 内联框架标签,用于向当前页面中引入一个其他页面
    属性:
  • src 指定要引入的网页的路径
  • frameborder 指定是否有边框
  • 引入的网页一般不能被爬取
  • 可以引入音频视频文件
audio标签 用来向页面引入音频
    属性:
  • controls 播放控制器/li>
  • autoplay 自动播放
  • loop 循环
  • source标签 嵌套在audio内,由浏览器选择自已支持的源
video标签用来向页面引入视频
    视频文件的引入 属性:
  • controls 播放控制器
  • autoplay 自动播放
  • loop 循环
